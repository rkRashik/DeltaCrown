### Recording Tips for Tutorials

1. **Preparation**: Have a test account with realistic data pre-loaded
2. **Resolution**: Record at 1920x1080, export at 1080p
3. **Browser**: Use a clean browser with no bookmarks bar / distracting tabs
4. **Mouse movement**: Move slowly and deliberately — viewers are following your cursor
5. **Pauses**: Pause on each important screen for 2-3 seconds before the narration continues
6. **Zoom**: Use OBS zoom filter or DaVinci Resolve zoom in post to highlight specific UI elements
7. **Dark mode vs Light mode**: Use whichever is the DeltaCrown default — keep consistent across all tutorials
8. **Tab discipline**: Only have the DeltaCrown tab open — no personal tabs visible
9. **Re-takes**: Record each section independently, not the whole video in one take
10. **Transitions**: Use consistent transitions — fade or quick cut, nothing fancy

### Narration Checklist

- [ ] Natural pace — no rushing
- [ ] Say the name of every button before clicking
- [ ] Repeat important information (account numbers, codes, etc.)
- [ ] Use "click" for buttons, "navigate" for page changes, "scroll" for scrolling
- [ ] Use present tense — "click Save" not "now you would click Save"
- [ ] Address the viewer as "you"
