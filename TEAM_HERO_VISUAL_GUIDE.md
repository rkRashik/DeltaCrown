## 🎭 Animation Guide

### Team Hub Animations

**Mesh Float** (20s loop):
```
0%   → translate(0, 0) rotate(0deg)
33%  → translate(2%, -2%) rotate(1deg)
66%  → translate(-2%, 2%) rotate(-1deg)
100% → translate(0, 0) rotate(0deg)
```

**Particles** (8-12s loop):
```
0%   → Invisible, start position
10%  → Fade in
50%  → Float up 60px, move right 30px
90%  → Still visible
100% → Fade out
```

**Pulse Circles** (4s loop):
```
0%   → scale(1), opacity(0.1)
50%  → scale(1.1), opacity(0.3)
100% → scale(1), opacity(0.1)
```

**Card Hover**:
```
Transform:  translateY(-4px)
Border:     Color → Accent
Shadow:     0 8px 24px rgba(99, 102, 241, 0.2)
Glow:       Appears (opacity 0 → 1)
```

### Team Detail Animations

**Logo Glow** (3s loop):
```
0%   → opacity(0.5), scale(1)
50%  → opacity(1), scale(1.1)
100% → opacity(0.5), scale(1)
```

**Stat Card Hover**:
```
Transform:  translateY(-4px)
Border:     rgba(255, 255, 255, 0.15) → #6366f1
Shadow:     0 8px 24px rgba(99, 102, 241, 0.3)
Duration:   0.3s ease
```

**Button Hover**:
```
Primary:    Darker gradient + lift
Secondary:  Brighter bg + border glow
Transform:  translateY(-2px)
Shadow:     Enhanced colored glow
```

## 🔧 Quick Customization

### Change Hub Accent Color
```css
/* teams-hub-hero.css */
:root {
  --hero-accent: #8b5cf6;  /* Change to purple */
}
```

### Change Detail Accent Color
```css
/* teams-detail-hero-v2.css */
:root {
  --team-accent: #f59e0b;  /* Change to amber */
}
```

### Adjust Animation Speed
```css
/* Faster particles */
.particle {
  animation: particleFloat 5s ease-in-out infinite;
}

/* Slower mesh */
.hero-gradient-mesh {
  animation: meshFloat 30s ease-in-out infinite;
}
```

### Add More Stats
Just duplicate the stat card HTML:
```html
<div class="hero-stat-card">
  <div class="stat-icon"><i class="fas fa-trophy"></i></div>
  <div class="stat-content">
    <div class="stat-number">25</div>
    <div class="stat-label">Tournaments</div>
  </div>
  <div class="stat-glow"></div>
</div>
```
