1. Identity Ownership

Core Rule: You may ONLY link game accounts that you personally own and have legitimate access to.

Prohibited activities:

  • Account Sharing: Using credentials from another person
  • Smurfing: Registering low-rank alternate accounts to exploit matchmaking
  • Boosting: Allowing another player to rank up your account
  • Account Trading: Buying, selling, or trading game accounts

Warning: By confirming ownership, you accept full responsibility for all activity on that game account.

2. Global Uniqueness Enforcement

Platform Rule: One game identity can only be linked to ONE DeltaCrown account ‚Äî forever.

Technical Implementation:

  • Database constraint enforces uniqueness on (game, identity_key)
  • Identity key is normalized and case-insensitive (e.g., "PlayerName#TAG" ‚Üí "playername:tag")
  • Attempting to link already-registered identity = immediate error

Blocked Action: "This Game ID is already linked to another account."

Why? Prevents identity theft, impersonation, and competitive fraud. If someone else registered your ID, contact support immediately.

4. Identity Lock Period

Security Measure: Core identity fields are locked for 30 days after creation.

What's locked:

  • Primary identity fields (Steam ID, Riot ID, PUBG UID, Konami ID, User ID, etc.)
  • Account-specific identifiers marked with icon

What you CAN update:

  • Rank, tier, division (as you climb/drop)
  • Region/server (if you move or switch data centers)
  • Visibility settings (Public/Protected/Private)
  • Pinned status (showcase on profile)

Rationale: Prevents rapid identity swapping to exploit tournaments, evade bans, or manipulate rankings.

6. Official Game Ban Policy

Zero Tolerance: Active bans from official anti-cheat systems result in immediate DeltaCrown suspension.

Monitored systems:

  • VAC (Valve Anti-Cheat) - Steam games
  • Riot Vanguard - VALORANT, League of Legends
  • BattlEye / EasyAntiCheat - PUBG, Apex, Fortnite
  • Konami Anti-Cheat - eFootball
  • Blizzard Warden - Overwatch, Call of Duty

Consequences:

  • Immediate account suspension (all features locked)
  • Forfeiture of all Delta Coins and pending earnings
  • Permanent ban if ban not overturned by game developer

Exception: If game developer overturns your ban, contact support with official proof to request reinstatement.
