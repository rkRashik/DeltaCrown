### Layers
```
Layer 3: Content (icon, name, count)
Layer 2: Dark gradient overlay (0.4 → 0.95 opacity)
Layer 1: Card image background
Layer 0: Border (primary color when active)
```

### Hover State
```
Background image zooms: scale(1.05)
Overlay lightens: (0.3 → 0.9 opacity)
Border appears if not active
Smooth 300ms transition
```

### Active State
```
Border: 2px solid #00ff88
Glow: 0 0 0 3px rgba(0, 255, 136, 0.1)
Name color: #00ff88
Count color: #00ff88
```

## Interaction Flow

### User Journey 1: Browse by Game
```
1. User lands on hub
   ↓
2. Scrolls past hero to games section
   ↓
3. Sees all 6 game cards with card images
   ↓
4. Hovers over VALORANT card (image zooms)
   ↓
5. Clicks VALORANT card
   ↓
6. Page reloads with ?game=valorant
   ↓
7. Tournament list filters to Valorant only
   ↓
8. VALORANT game card shows active state (green border)
```

### User Journey 2: Search
```
1. User sees search bar at top of games section
   ↓
2. Clicks search input (or presses Cmd/Ctrl + K)
   ↓
3. Types "winter"
   ↓
4. Clear button appears
   ↓
5. Presses Enter
   ↓
6. Page reloads with ?q=winter
   ↓
7. Tournament list filters to matches
```

### User Journey 3: Quick Register
```
1. User browses tournament cards
   ↓
2. Hovers over interesting tournament
   ↓
3. Card lifts up, shadow increases
   ↓
4. Banner image zooms slightly
   ↓
5. Dark overlay appears over image
   ↓
6. "Quick Register" button fades in
   ↓
7. User clicks Quick Register
   ↓
8. Taken to registration page
```
