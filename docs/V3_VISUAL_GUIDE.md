```
┌──────────┐      ┌──────────┐      ┌──────────┐
│          │      │          │      │          │
│ Browser  │─────▶│ Django   │─────▶│ Database │
│          │      │ API View │      │          │
│          │      │          │      │          │
└──────────┘      └──────────┘      └──────────┘
     │                 │                  │
     │            Response Cache          │
     │            (1-5 min TTL)           │
     │                 │                  │
     ▼                 ▼                  ▼
┌──────────┐      ┌──────────┐      ┌──────────┐
│ Client   │      │ Redis/   │      │ Postgres │
│ Cache    │      │MemCache  │      │          │
│ (1-2 min)│      │          │      │          │
└──────────┘      └──────────┘      └──────────┘
```

```
User Types in Search
        │
        ▼

        ▼

        │
        ▼
  Fetch Tournaments

        ▼
  Render Cards

## 🎭 Animation Timeline

### Page Load
```
0ms:    DOM Ready
100ms:  Hero fade in
300ms:  Stats cards slide in
500ms:  Tabs appear
700ms:  Content loads
900ms:  Animations complete
```

### Tab Switch
```
0ms:    Click detected
50ms:   Tab highlight
100ms:  Content fade out
200ms:  New content fade in
300ms:  Animation complete
```

### Infinite Scroll
```
User scrolls ▼
    │
    ▼

    │
    ▼
Show loading skeleton
    │
    ▼

    │
    ▼

    ▼
Animate cards in

## 🔧 Development Workflow

### Making Changes
```
1. Edit source files
   ├── static/js/tournaments-v3-*.js
   └── static/siteui/css/tournaments-v3-*.css

2. Collect static files
   $ python manage.py collectstatic --noinput

3. Clear cache (optional)
   $ python manage.py shell
   >>> from django.core.cache import cache
   >>> cache.clear()

4. Test in browser
   - Hard refresh (Ctrl+Shift+R)
   - Check console for errors
   - Test functionality

5. Commit changes
   $ git add .
   $ git commit -m "Your message"
```
