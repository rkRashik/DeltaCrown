## 🎭 Animations

### On Page Load
```
Hero Section:
- Grid pattern pulses
- Glowing orbs float
- Stats count up (0 → target)

Tournament Cards:
- Fade in (opacity: 0 → 1)
- Slide up (translateY: 30px → 0)
- Stagger delay (0.1s per card)
```

### On Scroll
```
Sections trigger when 10% visible:
[Section enters view]
  ↓
Elements fade in
  ↓
Staggered animation (0.1s delay each)
```

### On Hover
```
Cards:
  transform: translateY(-6px)
  box-shadow: larger + glow

Buttons:
  transform: translateY(-2px)
  box-shadow: enhanced
```

### On Filter
```
[User clicks filter]
  ↓
Hide non-matching (fade out 0.3s)
  ↓
Show matching (fade in with stagger)
  ↓
Update empty state if needed
```

## 🎯 Information Hierarchy

```
IMPORTANCE LEVELS:

CRITICAL (Hero Stats, Featured Tournament):
- Large text (42px+)
- Primary colors
- Prominent position
- Glow effects

HIGH (Tournament Cards, Game Selector):
- Medium text (20-28px)
- Clear contrast
- Interactive states
- Shadow effects

MEDIUM (Tournament Details, Metadata):
- Regular text (15-16px)
- Secondary colors
- Info grids
- Muted styling

LOW (Labels, Helper Text):
- Small text (12-14px)
- Muted colors
- Uppercase labels
- Minimal emphasis
```

## 🧭 User Journey Map

```
1. LAND ON PAGE
   ↓ (Hero Section)
   Sees: Live stats, premium design, clear value prop
   Action: Clicks "Browse Tournaments"
   
2. SCROLL DOWN
   ↓ (Featured Tournament)
   Sees: Main event spotlight, clear details
   Action: Clicks "View Details" or scrolls to see more
   
3. DISCOVER GAMES
   ↓ (Game Selector Grid)
   Sees: All available games with tournament counts
   Action: Clicks game to filter OR scrolls to see all
   
4. BROWSE TOURNAMENTS
   ↓ (Tournament Grid)
   Sees: All tournaments with rich details
   Action: Uses filters, clicks tournament card
   
5. LEARN PROCESS
   ↓ (How It Works)
   Sees: 4-step visual guide
   Action: Understands process, builds confidence
   
6. CONVERT
   ↓ (Call to Action)
   Sees: Final push to register or browse
   Action: Clicks CTA → Registration
```
